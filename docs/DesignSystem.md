# Design System & Color Tokens

The Clinical Digital Pathology Platform UI uses a strict color token palette and 8px grid spacing system.

---

## 🎨 Color Palette

| Token | Hex Value | Purpose |
| :--- | :---: | :--- |
| Background App | `#F8FAFC` | Platform canvas background |
| Background Card | `#FFFFFF` | Primary card container background |
| Primary Accent | `#2563EB` | Primary actions, interactive selections |
| Success | `#16A34A` | Verified states, low clinical risk badges |
| Warning | `#D97706` | Intermediate clinical risk, pending jobs |
| Error | `#DC2626` | High clinical risk, errors |
| Primary Text | `#111827` | Headings and primary text |
| Secondary Text | `#6B7280` | Subtitles and meta labels |
| Border Color | `#E5E7EB` | Clean structural dividers |

---

## 📐 Typography & Grid

- **Font Family**: Inter / Geist font stack.
- **Hierarchy Scale**: 64, 48, 36, 30, 24, 20, 18, 16, 14, 12.
- **Grid**: Strict 8px spacing system (`--space-8`, `--space-16`, `--space-24`, `--space-32`, `--space-48`).
