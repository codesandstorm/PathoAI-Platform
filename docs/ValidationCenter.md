# Scientific Validation Dashboard

The Validation Dashboard ([pathoai/dashboard/ui/app.js](file:///d:/Research/PathoAI-Platform/pathoai/dashboard/ui/app.js)) visualizes clinical agreement metrics (Intraclass Correlation Coefficient ICC, Bland–Altman mean bias & 95% limits of agreement, Pearson $r$, Spearman $\rho$, MAE, RMSE) and interactive agreement plots.
